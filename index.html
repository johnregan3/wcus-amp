Overview

This repo includes both a fully-functional plugin that demonstrates how to customize the default template provided by Automattic's AMP plugin, as well as many code samples showing how you can do this in your own projects.

This plugin is written for a planned talk at WordCamp US. Slides for this talk are available here.

Detail

The main plugin file, wcus-amp.php, contains documentation and basic functions to help add AMP support to your WordPress theme.

/samples is a directory of example usage of each piece of functionality, including some that aren't used in the plugin itself.
